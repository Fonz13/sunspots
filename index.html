How to "Jump" Locations

📱 On Mobile (Google Maps App)

  1. Tap any spot on the map.
  2. Scroll up the bottom panel to find the Plus Code (e.g., 83G4+QF Stockholm).
  3. Tap the code to copy it.
  4. Paste it into the search bar here!

Alternatively, tap the coordinates in the map drawer to copy them directly.

💻 On Desktop

Copy the full URL from your browser's address bar and paste it here, or use the plus code.
